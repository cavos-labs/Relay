# Relay

Landing page for **Relay** — Stellar smart accounts embedded in your product. Passkeys, gas abstraction, and programmable security. No extensions, seed phrases, or wallet popups.

---

## Tech stack

- **Next.js** (App Router) · **TypeScript** · **Tailwind CSS** · **Inter** (font)

---

## Prerequisites

- **Node.js** 18.x or later  
- **npm**

---

## Run locally

1. **Clone the repository** (if needed) and go to the project root:
   ```bash
   cd /path/to/Relay
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.

4. **Production build and run** (optional):
   ```bash
   npm run build
   npm start
   ```
